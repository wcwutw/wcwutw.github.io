2025/7/18 Astros @ Mariners

發表於2025-07-18|更新於2025-09-09|baseball

趁有空來補文章，就照時間最近的先補一補，不知道有生之年會不會寫完。

Box Score

太空人 1:6 水手 (F/9)

MLB.com wrap

Before Games

總之到了有棒球的地方怎麼可能不看(X，剛好我們來得這幾天的 free time 剛好有球賽，加上之前在說明會自介時有提到我會來看棒球，結果就半推半就地揪了一些人來看了ㄏ。我一開始就打算坐一壘上方，一來是這邊是最佳看城市景的角度（算是這個球場的特色吧），再來是我本來就沒有特別需要坐很前面的需求。但到了要揪人的時候大家就會有不同考量了，只能說一群人集體行動要顧慮到的東西還是比自己一個人可以到處亂跑還多很多。順帶一提我本來不知道這天會有 fireworks，所以也感謝 @Ray 說服我改成禮拜五來看，旅途中的回憶往往出於偶然吧！

Before Game Start

第一次到新球場我都喜歡提早很多到，這天大概提早了一個半小時，大約就是 Gate 開啟的時間左右。

從輕軌站拍過去

左外野可開合式的屋頂也算是這個球場的特色，雖然根據後來遊船導遊的說法一年根本不會用到幾次就是了XD。

從輕軌站往球場走最近的就是 CF Gate，順帶一提這裡安檢蠻鬆的，應該是我看過美國最鬆的地方（其實整個西雅圖的觀光景點安檢都不嚴格，相比我去過的其他城市）

外野牆上的名人堂，幾天後會多一個一朗

在外野合照

名人堂 countdown

牛棚 The Pen，看練投的部分 7/22 那篇有了，這天沒時間在賽前來看

本後景，工作人員沒有從一開始就擋，所以夠早進去是可以到內野座位區晃一下的

內野一壘側的景，個人感覺三樓比較壯觀一點

本後大門入口，風格跟大都會的很像

水手名人堂

最大的球隊商店，人多到不可思議，完全擠不進去，更不用說嘗試排隊買東西了，基本上如果有球賽應該很難在大商品部買，還好我在市區就有先去另一間官方商店買到一朗的紀念衣
